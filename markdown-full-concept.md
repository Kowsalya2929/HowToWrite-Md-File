# How to run and write md file :

## Steps:

1. Open VS Code
2. Open your .md file
3. Press: control + shift + v
4. Install Extention: Markdown Preview Enhanced
5. Right-click inside the Markdown file
6. Click Markdown Preview Enhanced: Open Preview to the Side
(or)
5. Press: control + shift + p
6. Type: Markdown Preview Enhanced: Open Preview to the Side
(or)
5. Press: control + shift + v

# Markdown Cheat Sheet

## 1. Ordered List

```js
1. Fist Point
2. Second Point
    1. Numbers
    1. Does not matter
```

### Output

1. Fist Point
2. Second Point
    1. First Number
    1. Second Number

## 2. Unordered List

```js
* Asterisks
* List
    * Nested
+ Can also use plus sign
- Can also use dash
```

### Output

* Asterisks
* List
    * Nested
+ Can also use plus sign
- Can also use dash


## 3. Block Quote

```js
>This is a quote
>
>>Nested Quote
>
>Hi quoute
>
>>Hi Nested quote
```

### Output

>This is a quote
>
>>Nested Quote
>
>Hi quoute
>
>>Hi Nested quote

## 4. Checklist

```js
- [ ] Must include space
- [x] Completed
```

### Output

- [ ] Must include space
- [x] Completed
